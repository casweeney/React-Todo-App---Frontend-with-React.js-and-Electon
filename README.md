# Todo App with React.js and  Electron

## Setup Procedures

### 1. Downlaod Repo
### 2. Open two (2) terminals
### 3. Navigate into project DIR using PC terminals (in my case it is todos)
### 4. In the first terminal run "npm start" this starts a react dev server on port 3000
### 5. In the second terminal run "npm run electron" this opens an electron app window which will similar to what you have in your broser window.
### 6. Sure you have downloaded the Backend Api for this project also which was built with Laravel. See link: https://github.com/casweeney/React-Todo-App---API-With-Laravel
