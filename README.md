# Todo App with React.js and  Electron

## Setup Procedures

### Downlaod Repo
### Open two (2) terminals
### Navigate into project DIR using PC terminals (in my case it is todos)
### In the first terminal run "npm start" this starts a react dev server on port 3000
### In the second terminal run "npm run electron" this opens an electron app window which will similar to what you have in your broser window.
